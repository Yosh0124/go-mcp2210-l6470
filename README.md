## Requirement

| Title | Content |
|---|---|
| OS | Windows 10 |
| Compiler | gcc |

## How to build

Download DLL files from [Microchip web site](https://www.microchip.com/wwwproducts/en/MCP2210) and save DLL file to root directory of this project.

And then, build exe file.

```
go build -o main.exe
```
